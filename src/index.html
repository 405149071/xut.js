时令缤纷冰草沙拉
try it

食材：冰草、时令蔬菜、水萝卜、彩椒、橄榄油红酒醋汁

做法：冰草最适合做沙拉，它清脆多汁的口感，配合夏季颜色缤纷的蔬菜，不仅视觉美观，营养均衡，酸甜的红酒醋汁更能凸显它的清爽。混合所有食材，拌入酱汁，冷藏1～2小时之后品尝效果更佳。

冰草大虾越南春卷
try it

食材：越南春皮卷、冰草、彩椒、去皮老虎虾

做法：越南春卷和大虾的软嫩口感，衬托出清脆的冰草和彩椒，加上酸甜的越南甜辣酱让口感层次更加丰富。非常适合作为开胃小食。冰草洗净，彩椒切片，老虎虾煮熟；将春卷皮泡软，晾干后，放入食材和虾，轻轻卷起；从中间切成2～3段即可。

豌豆冰草浓汤配云南羊奶酪
try it

食材：豌豆、冰草、薄荷叶、豌豆苗少许、羊奶酪少许

做法：豌豆带壳煮熟，去壳；加入薄荷叶继续煮5~10分钟，出锅放凉后；将汤和冰草加入搅拌机，加入盐和胡椒调味；用搅拌机将食材打碎成浓汤状；最后加入豆苗和羊奶酪装饰。冰草不易保存，但做成汤就不用担心它的外观了。冰草搭配新鲜的豌豆和清香的薄荷，非常健康，并含有大量的纤维素。
